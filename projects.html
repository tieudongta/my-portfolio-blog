Flowers Talk App

A responsive web application built using HTML, CSS, JavaScript, and various public APIs to explore an extensive collection of flowers along with their biological traits and cultural significance. Features include a “Flower of the Day,” personalized flower recommendations based on birthdate, interactive mini-games, and knowledge quizzes. This project showcases my front-end development skills, API integration, dynamic content rendering, and user-focused UI/UX design.

Vocab Fun App

A React-based web application designed to help users build English vocabulary through immersive reading and listening experiences. It integrates free resources from Project Gutenberg and LibriVox to provide access to public domain books with synchronized audio narration. Users can read along with real voice recordings, learn word meanings in context, and reinforce their vocabulary through interactive games. This project highlights my expertise in React, asynchronous data handling, third-party API integration, and building engaging educational tools.
