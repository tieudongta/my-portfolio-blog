Day 1: Callback Challenge Workbook

Posted on May 18, 2025

Our day 1 focused on:

  • forEach() - for printing/logging
  • map() - for transforming
  • Practice: convert to uppercase, appending strings, reformat data

Goal

Understand how to use forEach() to loop and print, and how to use map() to transform data without modifying the original array.

  1. Warm-Up: Names List

    const names = ["alice", "bob", "charlie", "diana"];

    Task 1A: Print each name with forEach()

    //Output:
    //Name: alice
    //Name: bob
    //...

    names.forEach(name => {console.log("Name:", name)});

  2. Uppercase with map()

    Task 2A: Create a new array of uppercase names

    //Output: ["ALICE", "BOB", "CHARLIE", "DIANA"]

    const upperNames = names.map(name => name.toUpperCase());
    console.log(upperNames);

  3. Add Prefix or Suffix with map()

    const animals = ["cat", "dog", "tiger"];

    Task3A: Add "Animal: " before each name

    //Output: ["Animal: cat", "Animal: dog", "Animal: tiger"]

    const modified = animals.map(animal => "Animal: "+animal);
    console.log(modified);

  4. Numbers: Square & Format

    const numbers = [2,4,6,8];

    Task 4A: Print each number with a message

    // Output:
    // The number is 2
    // ...

    const modified = numbers.map(number => "The number is " + number);
    console.log(modified);

    Task 4B: Create array of squares:

    //Output: [4, 16, 36, 64]

    const squares = numbers.map(number => number * number);
    console.log(squares);

  5. Real-World Formatting Example

    const users = [
    { first: "john", last: "doe" },
    { first: "jane", last: "smith" }
    ];

    Task 5A: Print full names

    // Output
    // john doe
    // jane
    // smith

    users.forEach(user =>{
    console.log(`${user.first} ${user.last}`);
    });

    Task 5B: Return capitalized full names

    // Output
    // ["John Doe", "Jane Smith"]

    const fullNames = users.map(user => {
    const cap = str => str[0].toUpperCase() + str.slice(1);
    return `${cap(user.first)} ${cap(user.last)}`;
    });

Bonus Challenge

const products = [

Tasks:

    // Output:

    products.forEach(p =>{
    console.log(p.name + ": $"+ p.price);

  2. Map to: "Notebook = $2.50" (Capitalized + formatted)

    // Output
    // ["Pen: $1.20",
    // "Notebook: $2.50",
    // "Pencil: $0.50"]

    const print = products.map(p => {
    const cap = str => str[0].toUpperCase()+str.slice(1);
    return `${cap(p.name)}: $${p.price.toFixed(2)}`;
    });
    console.log(print);
