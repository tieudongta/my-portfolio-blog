Applications built using only HTML, CSS, and vanilla JS

Posted on May 02, 2025

Practicing a vital step in any learning process. Building real, usable projects is the best way to solidify your knowledge and standing out with your portfolio. Here are 7 practical applications you can build using HTML, CSS, and vanilla JavaScript (no frameworks), each with ideas to stretch your skill.

1. Personal Finance Tracker

What it does: Allows users to log income and expenses, view totals, and categorize entries.

You'll learn:

  • DOM manipulation
  • Local storage
  • Date and currency formatting
  • Event handling

Stretch goals:

  • Filter by category/date
  • Export as CSV

Click here to learn more about this application

2. Responsive Weather App

What it does: Fetches and displays current weather based on user's location or search.

You'll learn:

  • Working with APIs (e.g. OpenWeatherMap)
  • Async/Await and Promises
  • Basic geolocation
  • Responsive design

Stretch goals:

  • Add 5-day forecast
  • Toggle between Celsius/Fahrenheit

Click here to learn more about this application

3. Quiz App

What it does: Presents multiple-choice questions, scores answers, and shows results.

You'll learn:

  • Data structure for questions
  • Event-driven programming
  • Real-time feedback

Stretch goals:

  • Add a timer
  • Store high scores in local storage

Click here to learn more about this application

4. Task Manager/To-Do List

What it does: Create, edit, delete, and complete tasks with optional deadlines.

You'll learn:

  • CRUD operations.
  • Storing and retrieving data from local storage
  • Creating reusable components

Stretch goals:

  • Drag-and-drop functionality
  • Task priority or tags

Click here to learn more about this application

5. Recipe App

What it does: Browse, search, and save cooking recipes.

You'll learn:

  • Working with an external API (e.g. Spoonacular)
  • Image and text rendering
  • Search/filter functionality

Stretch goals:

  • Save favorite recipes
  • Add your own recipes with image upload

Click here to learn more about this application

6. Portfolio Website(Self-contained App)

What it does: Showcases your projects, about info, and contact form (can be hosted)

You'll learn:

  • Clean layout and semantic HTML
  • Animations and transitions with CSS
  • Responsive design with Flexbox/Grid

Stretch goals:

  • Contact form with form validation
  • Add theme toggle (dark/light mode)

Click here to learn more about this application

7. Typing Speed Tester

What it does: Measures typing speed with real-time accuracy.

You'll learn:

  • Keyboard events
  • Timing functions
  • Realtime feedback logic

Stretch goals:

  • Save best scores
  • Random paragraph generation

Click here to learn more about this application

Final Thoughts

Start small and build your way up!

Thanks for reading. Good luck, and enjoy the journey!
