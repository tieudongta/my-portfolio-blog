To-Do List Tutorial Step 2

Posted on May 03, 2025

Tutorial Description

Let's continue building your own To-Do List App from step 1.In this step, we'll move on to adding the Mark As Complete and Deleting Tasks features. We'll improve the task list by making each task interactive.

Goals of this step

  1. Add a checkbox to mark tasks as completed
  2. Add a delete button to remove tasks.
  3. Update the rendering function to support these features.

Step 2-1: Modify function renderTasks() in script.js

Replace your existing renderTasks() with this version:


function renderTasks() {
  taskList.innerHTML = '';

  tasks.forEach((task, index) => {
    const li = document.createElement('li');

    // Checkbox for marking complete
    const checkbox = document.createElement('input');
    checkbox.type = 'checkbox';
    checkbox.checked = task.completed;
    checkbox.addEventListener('change', () => toggleComplete(index));

    // Span for task text
    const span = document.createElement('span');
    span.textContent = task.text;
    if (task.completed) {
      span.classList.add('completed');
    }

    // Delete button
    const deleteBtn = document.createElement('button');
    deleteBtn.textContent = '✖';
    deleteBtn.className = 'delete-btn';
    deleteBtn.addEventListener('click', () => deleteTask(index));

    li.appendChild(checkbox);
    li.appendChild(span);
    li.appendChild(deleteBtn);

    taskList.appendChild(li);
  });
}

Step 2-2: Add These Functions Below addTask()


function toggleComplete(index) {
  tasks[index].completed = !tasks[index].completed;
  saveAndRender();
}

function deleteTask(index) {
  tasks.splice(index, 1);
  saveAndRender();
}

function saveAndRender() {
  localStorage.setItem('tasks', JSON.stringify(tasks));
  renderTasks();
}

Step 2-3: Update CSS in style.css

Make completed tasks look crossed out:

.completed {
  text-decoration: line-through;
  color: gray;
}

What You Can Test Now:

  • Add a few tasks.
  • Check a box to mark them complete - the text strikes through.
  • Click the X to delete a task.
  • Refresh - your changes persist thanks to localStorage.
