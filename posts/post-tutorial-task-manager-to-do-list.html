To-Do List (Vanilla JS, HTML, CSS)

Posted on May 03, 2025

Project Description

A feature-rich, responsive To-Do List application built from scratch using only HTML, CSS, and Vanilla JavaScript. The app allows users to manage tasks efficiently with support for due dates, filtering, searching, sorting, drag-and-drop reordering, theme switching, and bulk actions.

Live Features

  • Add, edit, and delete tasks
  • Set optional due datesfor tasks
  • Mark tasks as complete/incomplete
  • Persistent storage withlocalStorage
  • Light/Dark Modetoggle
  • Filter tasks:all/active/completed
  • Search tasks by name
  • Sort tasksby due date
  • Bulk complete/delete selected tasks
  • Drag & drop task reordering
  • Responsive UI, mobile-friendly

Tech Stack

  • HTML5 - semantic structure
  • CSS3 - styling with Flexbox & dark mode
  • Vanilla JavaScript - logic, DOM manipulation, localStorage
  • No frameworks or libraries used

File Structure

/
	├── index.html       # App structure
	├── style.css        # All styling, including dark mode
	└── script.js        # Core logic (CRUD, filters, drag/drop, etc.)

Learning Outcomes

  • DOM manipulation & event handling
  • State management and localStorage
  • Building reusable, modular code
  • Responsive design principles
  • Clean UX with visual feedback
  • Progressive feature enhancement (e.g. drag-and-drop, filters)

Ideas for Further Enhancements

  • Task category/tags
  • Notifications or reminders
  • Backend integration(Node.js, Firebase)
  • User authentication & multiple lists
