Day 5 - Lookahead, Lookbehinds & Anchors

Posted on May 22, 2025

Welcome to Day 5 of your regex journey! Today we'll cover Lookahead, Lookbehinds, and Anchors - advanced tools that give you powerful matching logic without consuming characters. Let's stick to the proven format!

  1. ?= - Positive Lookahead

    Meaning:

    Matches only if what follows the current position matches a pattern, but doesn't consume it.

    /\w+(?=ing)/.exec("running"); // ['runn']
    //Checks that "ing" comes after, but only returns what's before it.

    Practice

    • Match words that are followed by "ed": /\w+(?=ed)/
    • Match any number followed by a %: /\d+(?=%)/

  2. ?! - Negative Lookahead

    Meaning:

    Matches only if what follows does not match the pattern.

    /\w+(?!ing)/.exec("runly"); // ['runly']

    Practice

    • Match words not ending in "ed": /\w+\b(?!ed)/
    • Match prices not followed by "$": /\d+(?!\$)/

  3. ?<= - Positive Lookbehind

    Meaning:

    Matches if the text before the current position matches a pattern (but does not consume it).

    /(?<=\$)\d+/.exect("$100"); // ['100']
    Matches a number preceded by a dollar sign.

    Practice:

    • Match digits preceded by #: /(?<=#)\d+/
    • Match username after "User: ": (?<=User:\s)\w+

  4. (?<!) - Negative Lookbehind

    Meaning:

    Matches if the text before the current position does not match a pattern.

    /(?<!\$)\d+/.exec("Price: 100"); // ['100']

    Practice

    • Match numbers not preceded by "$": (?<!\$)\d+

  5. Unicode Property Escapes (ES2018+)

    Meaning

    Match characters by Unicode properties.

    /\p{Script=Greek}/u.test("Ω"); // true (Greek letter omega)

    Practice

    • Match any Greek letter: /\p{Script=Greek}/u
    • Match any emoji:: /\p{Emoji}/u

Summary Table

Pattern Meaning Example Match
(?=abc) Positive lookahead Match if followed by "abc"
(?!abc) Negative lookahead Match if not followed by "abc"
(?<=abc) Positive lookbehind Match if preceded by "abc"
(?<!abc) Negative lookbehind Match if not preceded by "abc"
/\p{Script=Greek}/u Match any Greek letter Greek letter

Practice Challenges

  1. Match numbers followed by "kg" (e.g., 50kg)

    /\d+(?=kg)/

  2. Extract usernames after “User: ” (e.g., User: alice123)

    /(?<=User:\s)\w+/

  3. Match words not ending in “ed"

    /\b\w+\b(?!ed)/

  4. Match "pre" only if it's at the start of a word

    /\bpre/

  5. Match any emoji:

    /\p{Emoji}/u

Real World Practice

You'll use what you learned to solve real world task.

Data

const text = `
Usernames: alice_01, Bob99, _charlie, delta-4
Hex colors: #FFA07A, #0f0, #123abc
Greek letters: α, β, γ
Whitespace test: space, tab\t, newline\n
Transferred: $99
Balance: 1938
`;

Your Tasks

  1. Extract all usernames with only word characters and digits (no -)

    /\b\w+\b/g

  2. Match all hex color digits

    /[0-9a-fA-F]+/g

  3. Find Greek letters

    /\p{Script=Greek}/gu

  4. Match all whitespace sequences

    /\s+/g

  5. Extract all prices after the dollar sign

    /(?<=\$)\d+/g

Day 5 Quiz

  1. What does the regex /\d+(?=%)/ >the string "50% off"

    1. "50%"
    2. "50"
    3. "%"
    4. No match

    Explanation:
    (?=%) is a positive lookahead checking that digits are followed by %, but it only matches the digits, not the %

  2. Which of the following matches a number not preceded by $?

    1. /\$(?!\d+)/
    2. /(?<!\$)\d+/
    3. /\d+(?<!\$)/
    4. /\d+(?=\$)/

    Explanation
    (?<!\$) is a negative lookbehind ensuring the number is not immediately preceded by a $

  3. What does the regex /\bwin\b/ NOT match?

    1. "win"
    2. "window"
    3. "to win"
    4. "win-win"

    Explanation:
    \b matches word boundaries, so /\bwin\b/ matches "win" as a whole word, but not inside "window"

  4. Which of these matches the word “pre” only if it's at the start of a word?

    1. /pre/
    2. /\bpre/
    3. /pre\b/
    4. /\Bpre/

    Explanation:
    \b asserts the position at a word boundary, so this matches "pre" only at the start of words.

  5. What does /\w+(?!ing)/ match?

    1. Words ending with “ing”
    2. Words not ending with “ing”
    3. Only “ing”
    4. Nothing

    Explanation:
    (?!ing) is a negative lookahead ensuring the word is not followed by "ing".

  6. What does /(?<=\$)\d+/ match in "$250 is the price"?

    1. "250"
    2. "$250"
    3. "is"
    4. No match

    Explanation:
    Positive lookbehind (?<=\$) matches digits only if preceded by a $, but doesn't include the $ itself.

  7. What is the purpose of \B?

    1. Match word boundaries
    2. Match non-word boundaries
    3. Match whitespace
    4. Match digits

    Explanation:
    Opposite of \b, \B matches positions inside words (no word boundary).

  8. Which regex matches numbers followed immediately by “kg”?

    1. /\d+(?=kg)/
    2. /\d+(?!kg)/
    3. /\d+(?<=kg)/
    4. /kg(?=\d+)/

    Explanation:
    Positive lookahead checks for "kg" after the digits, but only matches the digits themselves.

  9. Which regex uses a negative lookbehind?

    1. /\d+(?<!\$)/
    2. /\d+(?=\$)/
    3. /(?<!\$)\d+/
    4. /\$(?!\d+)/

    Explain:
    (?<!\$) is the negative lookbehind that checks if the preceding character is not $.

  10. Which flag is required to use Unicode property escapes like \p{Script=Greek} ?

    1. "g"
    2. "i"
    3. "u"
    4. "m"

    Explain:
    The u flag enables Unicode mode, necessary for property escapes.
