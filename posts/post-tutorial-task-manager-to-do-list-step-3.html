To-Do List Tutorial Step 3

Posted on May 03, 2025

Tutorial Description

Let's continue building your own To-Do List App from step 2.In this step, we'll implement the feature that allows you to directly edit text of the task in the list.

Step 3-1: Modify Task Rendering in function renderTasks()

Replace the part where you create the <span> for the task test with this:


// Editable span
const span = document.createElement('span');
span.textContent = task.text;
span.contentEditable = true;
if (task.completed) {
  span.classList.add('completed');
}

// Save changes when user finishes editing
span.addEventListener('blur', () => editTask(index, span.textContent));
span.addEventListener('keydown', (e) => {
  if (e.key === 'Enter') {
    e.preventDefault();
    span.blur(); // triggers blur event
  }
});

Step 3-2: Add The editTask() Function


function editTask(index, newText) {
  newText = newText.trim();
  if (newText) {
    tasks[index].text = newText;
    saveAndRender();
  } else {
    deleteTask(index); // optional: delete if emptied
  }
}

Step 3-3: Update CSS in style.css

You can add a light outline to show when you're editing the task text:

li span[contenteditable="true"]:focus {
  outline: 2px dashed #007bff;
  background-color: #eef;
}

Try It Out

  • Click any task text to edit it.
  • Press Enter or click out side to save.
  • Leave the text empty and blur -> it deletes the task.
