To-Do List Tutorial Step 6

Posted on May 03, 2025

Tutorial Description

Let's continue building your own To-Do List App from step 5.In this step, we'll implement the feature that allows you to Sort Tasks by Due Date- this will let users view taks in order of upcoming deadlines.

Goals

  1. Add a dropdown to sort tasks by due date.

Step 6-1: Add a Sort Dropdown in HTML

Place it near your search input(above the task list):


<select id="sort-select">
  <option value="default">Sort by...</option>
  <option value="due-date">Due Date</option>
</select>

Add Some CSS:

#sort-select {
  width: 100%;
  padding: 0.5rem;
  margin-bottom: 1rem;
  border-radius: 4px;
  border: 1px solid #ccc;
}

Step 6-2: Track Sorting Method in JS

At the top of script.js, add:


let sortMethod = 'default';

Step 6-3: Add a Listener to the Dropdown

Below renderTasks():


document.getElementById('sort-select').addEventListener('change', (e) => {
  sortMethod = e.target.value;
  renderTasks();
});

Step 6-4: Update renderTasks() to Sort

Before looping through tasks in renderTasks():


let sortedTasks = [...tasks]; // clone array

if (sortMethod === 'due-date') {
  sortedTasks.sort((a, b) => {
    if (!a.dueDate) return 1;
    if (!b.dueDate) return -1;
    return new Date(a.dueDate) - new Date(b.dueDate);
  });
}

Use sortedTasks.forEach(....) instead of tasks.forEach(...)
