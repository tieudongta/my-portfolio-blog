Tutorial Description

Let's continue building your own To-Do List App from step 7.In this step, we'll implement the feature that allows you to Toggle Theme (Light/Dark Mode).

Goals

Update the button Dark Mode so that it enables toggling between light and dark themes. This includes:

  • Apply a class to the <body> to switch themes.
  • Storing the user's choice in localStorage so it persists.
  • Styling both themes with CSS.

Step 8-2: Add Theme Style in CSS

Add these styles to your CSS file:


:root {
  --bg-color: #fff;
  --text-color: #111;
  --task-bg: #f9f9f9;
}

body.dark-theme {
  --bg-color: #1e1e1e;
  --text-color: #f1f1f1;
  --task-bg: #2c2c2c;
}

body {
  background-color: var(--bg-color);
  color: var(--text-color);
  transition: background-color 0.3s, color 0.3s;
}

li {
  background: var(--task-bg);
  transition: background 0.3s;
}

#theme-toggle {
  position: absolute;
  top: 1rem;
  right: 1rem;
  background: none;
  border: none;
  font-size: 1.5rem;
  cursor: pointer;
}

Step 8-3: Add Theme Logic in script.js

Add the below code:


const themeToggleBtn = document.getElementById('theme-toggle');

// Load saved theme preference
function loadTheme() {
  const savedTheme = localStorage.getItem('theme');
  if (savedTheme === 'dark') {
    document.body.classList.add('dark-theme');
  }
}

// Toggle theme
function toggleTheme() {
  document.body.classList.toggle('dark-theme');
  const theme = document.body.classList.contains('dark-theme') ? 'dark' : 'light';
  localStorage.setItem('theme', theme);
}

themeToggleBtn.addEventListener('click', toggleTheme);
loadTheme(); // Apply theme on load
