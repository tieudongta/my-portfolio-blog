To-Do List Tutorial Step 5

Posted on May 03, 2025

Tutorial Description

Let's continue building your own To-Do List App from step 4.In this step, we'll implement the feature that allows you to Filter and Search Tasks- this feature helps users narrow down tasks by status or keywords.

Goals

  1. Filter tasks by: All, Active, Completed.
  2. Search tasks by keyword.

Step 5-1: Add Filter Buttons in index.html

Inside your container (typically after the task list), add:

<div class="filters">
  <button onclick="setFilter('all')">All</button>
  <button onclick="setFilter('active')">Active</button>
  <button onclick="setFilter('completed')">Completed</button>
</div>

Step 5-2: Track Filter and Search Query in JS

At the top of script.js, add:


let currentFilter = 'all';
let searchQuery = '';

Step 5-3: Add function setFilter() and updateFilterButtons()


function setFilter(filter) {
  currentFilter = filter;
  renderTasks();
}

function updateFilterButtons() {
  const buttons = document.querySelectorAll('.filters button');
  buttons.forEach(button => {
    button.classList.toggle('active', button.textContent.toLowerCase() === currentFilter);
  });
}

Step 5-4: Add Search Input to HTML

Right above the task list:

And style it in CSS:


#search-input {
  width: 100%;
  padding: 0.5rem;
  margin: 1rem 0 0.5rem;
  border: 1px solid #ccc;
  border-radius: 4px;
}

Step 5-5: Handle Search Input in JS

document.getElementById('search-input').addEventListener('input', (e) => {
  searchQuery = e.target.value;
  renderTasks();
});

Step 5-6: Update renderTasks() to Apply Filter & Search

Inside your forEach loop in renderTasks():


if (
  (currentFilter === 'active' && task.completed) ||
  (currentFilter === 'completed' && !task.completed) ||
  (searchQuery && !task.text.toLowerCase().includes(searchQuery.toLowerCase()))
) {
  return;
}

Try It Out

  • Filter between all/active/completed tasks.
  • Instantly search by text.
