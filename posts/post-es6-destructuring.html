ES6: Destructuring in JavaScript

Posted on May 23, 2025

Destructuring in JavaScript is a powerful feature introduced in ES6 (ECMAScript 2015) that allows you to unpack values from arrays or properties from objects into distinct variables in a concise and readable way. Let's break it down comprehensively:

  1. Array Destructuring

    Basic syntax:

    const numbers = [1,2,3];
    const [a,b,c] = numbers;
    console.log(a);// 1

    Skipping elements:

    const [first, , third] = [10,20,30];
    console.log(third); // 30

    Using rest operator

    const [head, ...tail] = [1,2,3,4];
    console.log(tail); // [2,3,4]

    Default values:

    const [x=5, y=7] = [1];
    console.log(x,y); // 1, 7

  2. Object Destructuring

    Basic syntax:

    const person = { name: "Alice", age: 23};
    const {name, age} = person;
    console.log(name); // "Alice"

    Renaming variables:

    const { name: fullName, age: years } = person;
    console.log(fullName); // "Alice"

    Default values:

    const {nickname = "No nickname"} = person;
    console.log(nickname);// "No nickname"

    Nested objects:

    const user = {
    id: 1,
    info:
    username: "jdoe",
    email: "jdoe@example.com"
    }
    };
    const { info: { username, email}} = user;
    console.log(username); // "jdoe"

  3. Function Parameters Destructuring

    For Arrays:

    function sum([a,b]) {
    return a + b;
    }
    console.log(sum([3,7])); // 10

    For Objects:

    function greet({ name, age }) {
    return `Hello, ${name}. You are ${age}.`;
    }
    console.log(greet({ name: "Alice", age: 30}));

  4. Destructuring with Loops

    const users = [
    { id: 1, name: "Tom" },
    { id: 2, name: "Jerry" }, ];
    for (const {id, name } of users) {
    console.log(`${id}: ${name}`);
    }

  5. Practical Use Cases

    • Extracting props in React components
    • Parsing JSON responses
    • Returning multiple values from a function
    • Handling function arguments with defaults

  6. Destructuring Gotchas

    Be careful with undefined

    const [x] = [];
    console.log(x); // undefined
    const { y } = {}; console.log(y); // undefined

    Renaming + default:

    const { name: fullName = "Guest" } = {};
    console.log(fullName); // "Guest"

  7. Destructuring + Spread & Rest

    const person = { name: "Alice", age: 30, city" "NY" };
    const { name, ...rest } = person;
    console.log(rest); // { age: 30, city: "NY" }

  8. Summary

    Concept Syntax Example
    Array Destructuring const [a, b] = [1, 2];
    Object Destructuring const { name } = user;
    Default Values const [x = 10] = [];
    Rest Pattern const [head, ...tail] = list;
    Nested Structures const { a: { b} } = obj;
    Param Destructuring function({id, name}) {}
